# ⌛ Focus Timer

Segundo desafio proposto no módulo 5 do curso Explorer da Rocketseat

<br>
<br>

## ⚙️ Funcionalidades

- Temporizador
- Tocar som relaxante (cards que se encontra ao lado direito)
- Para iniciar o som de um card, tem que parar o som do card anterior

<br>
<br>
